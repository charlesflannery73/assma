API Quick start guide

List all assets
curl -s -k -H "Authorization: Token your-api-key" -H "content-type: application/json" https://hostname:port/api/v1/asset/

List all orgs
curl -s -k -H "Authorization: Token your-api-key" -H "content-type: application/json" https://hostname:port/api/v1/org/

List one asset - by asset_id
curl -s -k -H "Authorization: Token your-api-key" -H "content-type: application/json" https://hostname:port/api/v1/asset/1/

List one org by org id
curl -s -k -H "Authorization: Token your-api-key" -H "content-type: application/json" https://hostname:port/api/v1/org/1/

Search orgs (exact match) by name, id, level, tier, (partial match) by name_like, comment
curl -s -k -H "Authorization: Token your-api-key" -H "content-type: application/json" https://hostname:port/api/v1/org/?name=1.2.3.4

Search assets (exact match) by org, org_id, name, id, type, (partial match) by org_like, name_like, comment
curl -s -k -H "Authorization: Token your-api-key" -H "content-type: application/json" https://hostname:port/api/v1/asset/?type=ipv4

Create new org
curl -X POST -s -k -H "Authorization: Token your-api-key" -H "content-type: application/json" http://127.0.0.1:8000/api/v1/org/ -d '{"name":"new org 25", "level": "National", "sector": "ICT", "Tier": 5, "comment": "hi"}'

Create new asset (must know the org_id)
curl -X POST -s -k -H "Authorization: Token your-api-key" -H "content-type: application/json" http://127.0.0.1:8000/api/v1/asset/ -d '{"name":"8.8.8.8", "type":"ipv4", "org":5}'

Delete org by org id (successful delete returns a 204 No Content)
This will fail if the org has assets related to it. Assets related to an org must be deleted first before the org will delete successfully
curl -X DELETE -s -k -H "Authorization: Token your-api-key" -H "content-type: application/json" http://127.0.0.1:8000/api/v1/org/5/

Delete asset by asset id (successful delete returns a 204 No Content)
curl -X DELETE -s -k -H "Authorization: Token your-api-key" -H "content-type: application/json" http://127.0.0.1:8000/api/v1/asset/7/
